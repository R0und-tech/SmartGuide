# SmartGuide: голосовой помощник для слепых и слабовидящих

[![Python](https://img.shields.io/badge/Python-3.9%2B-blue.svg)](https://www.python.org/)
[![Raspberry Pi](https://img.shields.io/badge/Raspberry%20Pi-Zero%20W-green.svg)](https://www.raspberrypi.org/)

## Описание

**SmartGuide** — компактный голосовой помощник на базе **Raspberry Pi Zero W**, который помогает слепым и слабовидящим ориентироваться в пространстве.

При движении устройства (ходьба, поворот) система:
1. Определяет движение с помощью модуля **GY-521** на базе микросхемы **MPU6050**.
2. Делает фото с помощью встроенной камеры.
3. Отправляет изображение в **GigaChat** для анализа.
4. Получает текстовое описание сцены.
5. Озвучивает описание c помощью **Yandex SpeechKit API**.

Работает автономно от батареи, с автоподключением Bluetooth-наушников.

---

## Функции

- Детекция движения (акселерометр + гироскоп)
- Автоматическая съёмка при активности
- Анализ изображения через GigaChat
- Озвучивание через Yandex SpeechKit
- Экономия батареи: активация только при движении
- Поддержка Bluetooth-наушников

---

## Требования

### Аппаратная часть
- Raspberry Pi Zero W
- MPU6050 (I2C)
- USB Camera Module
- Bluetooth-наушники (A2DP)
- PowerBank 5000+ мАч

### Программная часть
- Raspberry Pi OS
- Python 3.9+
- Включённые интерфейсы: I2C, Camera
- Доступ в интернет (для GigaChat и Yandex SpeechKit)

## Конфигурация

1. **GigaChat API**  
   В `main.py` укажи свой ключ:
   ```python
   GIGACHAT_CREDENTIALS = "твой_ключ"
   ```

2. **Yandex SpeechKit**  
   В `main.py` укажи IAM-токен или OAuth:
   ```python
   YANDEX_IAM_TOKEN = "твой_IAM_токен"
   ```
